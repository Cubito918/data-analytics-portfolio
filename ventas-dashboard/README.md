# Dashboard de Ventas — Python y Visualización

Proyecto de Data Analytics enfocado en la creación de un dashboard para analizar el rendimiento de ventas por producto y mes.

## Objetivo
Visualizar datos de ventas para facilitar la toma de decisiones comerciales.

## Herramientas
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Visualizaciones
- Ventas por producto
- Ventas por mes
- Identificación de tendencias

## Insights
- Laptop es el producto con mayor ingreso.
- Abril presenta el pico más alto de ventas.
- Se recomienda reforzar campañas en meses de alta demanda.
