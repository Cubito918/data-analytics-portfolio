#  Análisis de Ventas — Python & Pandas

Proyecto de análisis de ventas usando Python para identificar:
- Productos más vendidos
- Meses con mayor ingreso
- Clientes de alto valor

##  Herramientas usadas
- Python
- Pandas
- Jupyter Notebook

##  Objetivo
Obtener insights comerciales para mejorar decisiones de negocio.

##  Insights principales
- La Laptop es el producto con mayores ventas.
- Abril es el mes con mayor ingreso.
- El cliente top genera alto impacto económico.
